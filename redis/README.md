# go-redis

[go-redis](https://github.com/go-redis/redis): go-redis 是一个用于 Go 语言的 Redis 客户端库, 提供了丰富的功能, 包括连接池管理, 事务支持, 发布/订阅, Lua 脚本执行等. 它支持 Redis 的多种数据类型, 如字符串, 哈希, 列表, 集合, 有序集合等, 并且可以与 Redis 的哨兵模式, 集群模式无缝集成.
