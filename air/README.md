# air

[Air](https://github.com/air-verse/air) 是为 Go 应用开发设计的另外一个热重载的命令行工具.只需在你的项目根目录下输入 air, 然后把它放在一边, 专注于你的代码即可.

注意: 该工具与生产环境的热部署无关.
