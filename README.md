# xorm

[xorm](https://gitea.com/xorm/xorm): xorm 是一个简单但强大的 Go 语言 ORM 库, 使用它可以大大简化我们的数据库操作.

- `Struct <-> Table`映射支持.
- 可链式调用的 API.
- 事务支持.
- 同时支持 ORM 和原生 SQL 操作.
- 数据库模式同步支持.
- 查询缓存加速.
- 通过`xorm.io/reverse`提供数据库反向生成支持.
- 简单的级联加载支持.
- 乐观锁支持.
- 通过`xorm.io/builder`提供 SQL 构建器支持.
- 自动读写分离.
- Postgres 模式支持.
- 上下文缓存支持.
