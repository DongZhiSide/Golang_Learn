# excelize

[Excelize](https://github.com/xuri/excelize) 是一个纯 Go 编写的库, 提供一组函数, 允许你写入和读取`XLAM, XLSM, XLSX, XLTM, XLTX`文件.

支持阅读和编写由`Microsoft Excel™ 2007`及以后版本生成的电子表格文档. 通过高兼容性支持复杂组件, 并提供了流式 API, 用于从包含大量数据的练习表中生成或读取数据. 该库需要 Go 版本 1.24.0 或更高版本.
